# Atividade

Tarefa do dia 1, 2 e 3: Com base no conhecimento adquirido em relação a versionamento com Git, você deverá criar um repositório para ser utilizado daqui para frente, e neste repositório deverá subir todos arquivos importantes das Sprints anteriores, como arquivos de planejamento de testes, mapas mentais, collections Postman entre outros. 

O repositório deverá ser versionado por branches, e os arquivo do PB devem ficar dentro de uma pasta chamada "Documentos". Deve ser criado um README dando uma breve introdução ao repositório.

Modalidade: Individual

**Local/Formato da entrega da atividade: Link do repositório Github deverá ser postado em uma thread única no canal Geral**
