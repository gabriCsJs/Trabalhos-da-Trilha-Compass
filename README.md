# Proposito
Este repositorio tem como objetivo centralizar e versionar todos os arquivos gerados ao longo das sprints do programa AWS & AI for software quality enginnering.

# Padronizaçao
O versionamento sera realizado por branches onde a branch principal sera a branch ```main```
Cada semana sera uma branch e ao final da semana na sexta-feira sera realizado o ```merge``` com a branch ```main```

# Estrutura de organizaçao
```
Documentos
 ┣ Fase_X
 ┃ ┣ Sprint_Y
 ┃ ┃ ┣ Semana_Z
 ┃ ┃ ┃ ┣ Resultados
 ┗ README.md
```
